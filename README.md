# 📱 Mobile Price Classification

## 📌 Proje Hakkında  
Bu proje, mobil cihazların teknik özelliklerine dayanarak fiyat aralıklarını tahmin etmeyi amaçlayan bir makine öğrenmesi sınıflandırma projesidir.

## 🧠 Kullanılan Yöntemler  
- Classification Algorithms  
- Feature Engineering  
- Train/Test Split  

## 🛠️ Teknolojiler  
Python, Pandas, NumPy, Matplotlib, Scikit-learn  

## 🎯 Amaç  
Mobil cihaz özelliklerine göre fiyat segmentlerini doğru şekilde sınıflandırmak.  

## 📈 Model Performansı  
Model, veri setindeki özellikleri analiz ederek test verileri üzerinde başarılı sınıflandırma sonuçları elde etmiştir. *(Accuracy eklersen çok daha güçlü olur)*
